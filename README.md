# replace.sh

Usage:
  $0 [-R] [-e] directory|file oldChars newChars

Description:
  Repalce all occurrences of oldChar in the file or in the files of the directory with newChar.

Options:
  -R:find files in the directory recursively
  -e:enable regular expression
